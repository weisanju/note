## 拉取远程分支合并到本地分支

```shell
git pull origin feature/develop-4.4.0-public
git pull upstream feature/develop-4.4.0-public
```



## 删除未加入到版本管理的文件

```shell
git clean -fd
```



## 将版本库里的head替换工作区与暂存库

```shell
git checkout head . 
```







### 不分页

```
git config --global core.pager cat 
```

### 分页

```
git config --global core.pager less
```
