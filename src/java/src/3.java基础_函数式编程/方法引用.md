# 方法引用

方法引用 可以看作是 一个匿名内部类的实现,实质上是一个 匿名内部类的对象



# 方法引用有四种

**指向静态方法的引用**

**指向某个对象的实例方法的引用**

**指向某个类型的实例方法的引用**

**指向构造方法的引用**



| 方法 引用                                   | lambada表达式          |
| ------------------------------------------- | ---------------------- |
| *String::valueof*  (静态方法)               | *x->String.valueof(x)* |
| *Object::toString()* (对象的实例方法的引用) | *x->x.toString();*     |
| *x::toString()* (类型的实例方法的引用)      | *x->x.toString();*     |
| *ArrayList::new* (构造方法的引用)           | *()->new ArrayList()*  |
